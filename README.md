# frontend-components

## Frontend Components

Library of reusable components for web interface development.

The goal of this repository is to centralize frontend components, allowing reuse across different applications, reducing code duplication and maintaining visual and technical consistency.

## Objectives

- Component reuse
- UI standardization
- Low coupling
- Easy maintenance
- Project independence

## Stack

- HTML5
- CSS3
- JavaScript (ES6+)

Specific components may have implementations for:

- Angular
- React
- Vue
